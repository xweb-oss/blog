## slot
1. 单个插槽
父组件
```html
<template>
  <div class="first-page">
    <div class="first-content">
        换个名字,来叫我。
      <child>
          <div>
              jdsjdjsdjsjd
          </div>
          <span style="color: red;">嗯哼?</span>
      </child>
  </div>
  </div>
</template>
<script type="text/javascript">
    import child from './my_child.vue';
    export default {
        name: 'parent',
        components:{
          // 用来声明页面组件  
          child
        }
    }
</script>
```
子组件
```html
<template>
    <div class="sidebar">
        <h1>单个插槽</h1>
        <div class="slot">
           <slot></slot>
        </div>
        <h1>单个插槽6666</h1>
    </div>
</template>
<script>
    export default {
        data() {
            return {}
        },
        computed:{}
    }
</script>

<style scoped>
    .sidebar{
        text-align: left;
    }
</style>
```
预览
` 换个名字,来叫我。`
`单个插槽`
`jdsjdjsdjsjd`
`嗯哼?`
`单个插槽6666`

当把子组件中的`  <div class="slot"><slot></slot></div>`注释。那么`jdsjdjsdjsjd` `嗯哼?`将不会显示。

2. 具名插槽 name - string，用于命名插槽。
区别于匿名插槽，当存在多个插槽的时候，当我们向在不同的插槽插入不同的内容，我们就需要用到匿名插槽    

父组件
```html
<template>
  <div class="first-page">
    <div class="first-content">
        换个名字,来叫我。
      <child>
          <span style="color: red;">嗯哼?</span>
           <!-- 替换子组件匿名插槽 -->
           <span slot="name_slot" style="color: #11caff;">我将替换子组件slot name="name_slot"的插槽</span>
          <!-- 替换子组件slot name="name_slot"的插槽 -->
      </child>
  </div>
  </div>
</template>
<script type="text/javascript">
    import child from './my_child.vue';
    export default {
        name: 'parent',
        components:{
          // 用来声明页面组件  
          child
        }
    }
</script>
```
子组件
```html
<template>
    <div class="sidebar">
        <h1>单个插槽</h1>
        <div class="slot">
           <slot></slot>
        </div>
        <div class="nameSlot">
            <slot name="name_slot"></slot>
        </div>
        <h1>单个插槽6666</h1>
    </div>
</template>
<script>
    export default {
        data() {
            return {}
        },
        computed:{}
    }
</script>

<style scoped>
    .sidebar{
        text-align: left;
    }
</style>
```
